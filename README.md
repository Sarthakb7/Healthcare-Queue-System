# Healthcare Queue Management System

Healthcare Queue Management System is a web application developed to reduce waiting time and improve patient flow in hospitals. 
The system allows patients to register online, generate tokens, track queue status, and view estimated waiting times.


## Features
* Patient Signup and Login
* Patient Registration
* Token Generation
* Queue Status Tracking
* Admin Queue Management
* Hospital Dashboard


## Tech Stack
* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express.js
* Database: MongoDB
